## Install
```sh
# clone the repository
git clone https://github.com/aleksandrchusovitin/frontend-project-lvl1.git

# go to the folder with the repository
cd frontend-project-lvl1

# install dependencies
make install

# install the package
make link
```

## Usage example
```sh
# choose from five games
brain-calc

brain-even

brain-gcd

brain-prime

brain-progression
```
